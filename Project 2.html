Project Title

This project included the design and modelling of signal processing PCB to enable efficient communication between the power converter provided and the DSP. Both open loop and closed loop operations were simulated, implemented and tested for validation.
In this project, we designed the control and monitoring of a three phase RL load using texas instrument digital signal processor (TMS320F28335). The image depicts an overview of the overal work done.

Hardware in Loop implementation with RTBOX, Simulink and Gausch Converter, Gausch Converter, TI digital signal processor
